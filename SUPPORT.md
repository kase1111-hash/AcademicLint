## Getting Help

### GitHub Discussions

For questions, ideas, and general discussion:

**[GitHub Discussions](https://github.com/kase1111-hash/academiclint/discussions)**

Use Discussions for:
- Questions about usage or configuration
- Feature ideas and suggestions
- Sharing how you use AcademicLint
- Community discussion

### GitHub Issues

For bugs and feature requests:

**[GitHub Issues](https://github.com/kase1111-hash/academiclint/issues)**

Use Issues for:
- Bug reports (use the bug report template)
- Feature requests (use the feature request template)
- Documentation issues

Before opening an issue:
1. Search existing issues to avoid duplicates
2. Use the appropriate issue template
3. Provide as much detail as possible

### Email

For private inquiries or matters not suitable for public discussion:

**Email:** kase1111@gmail.com

## Security Issues

**Do NOT report security vulnerabilities through public channels.**

Please see [SECURITY.md](./SECURITY.md) for instructions on reporting security issues responsibly.

## Response Times

- **GitHub Discussions**: Community-driven; maintainers check periodically
- **GitHub Issues**: We aim to triage within 7 days
- **Email**: We aim to respond within 5 business days
- **Security Issues**: Acknowledged within 48 hours (see [SECURITY.md](./SECURITY.md))

## Code of Conduct

We expect all participants to be respectful and constructive. Please be kind and patient with maintainers and other community members.
